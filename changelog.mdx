## Recent Updates

Stay informed about Pipeops releases. You receive notifications for major updates through your dashboard, and check this page for detailed changelogs. Each release includes new features, framework support, bug fixes, and performance improvements.

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added support for deploying Golang applications with one-click templates.
- Introduced real-time deployment logs for better monitoring.
- New webhook integration for custom notifications.

## Improvements

- Enhanced dashboard performance by `>30%` for faster page loads.
- Updated UI components with improved accessibility.

## Bug Fixes

- Fixed intermittent deployment failures on high-traffic projects.
- Resolved configuration sync issues with Git repositories.

</Update>

<Update label="2024-09-20" description="v1.1.0" tags={["feature", "bugfix"]}>

## New Features

- Native support for Django templates with auto-environment setup.
- Added Laravel deployment presets for PHP projects.

## Bug Fixes

- Corrected Vue.js build errors during static site generation.
- Fixed Next.js server-side rendering compatibility issues.

## Improvements

- Streamlined React.js deployments with optimized caching.

</Update>

<Update label="2024-08-10" description="v1.0.0" tags={["feature", "breaking"]}>

## New Features

- Core deployment engine for React.js, Vue.js, Next.js, PHP, and Laravel.
- Custom domain management and SSL auto-provisioning.
- Git-based continuous deployment workflows.

## Breaking Changes

- Updated API authentication to use `{API_KEY}` headers exclusively.

## Initial Setup Example

To upgrade to v1.0.0, update your project settings:

````bash
pipeops deploy --version 1.0.0 --token YOUR_TOKEN
````

</Update>
