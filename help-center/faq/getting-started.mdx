## Quick Start

To begin using Pipeops:

1. Sign up at `https://app.pipeops.com`
2. Create your first project in the dashboard
3. Connect your GitHub or GitLab repository
4. Click "Deploy" to trigger your first deployment

## Need Help?

Check docs for guides and visit our community if you get stuck.
